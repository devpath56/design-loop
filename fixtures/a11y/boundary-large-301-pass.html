Contrast boundary probe: large text, 3.01:1

Large-scale heading copy at 26px regular weight. Measured contrast ratio is 3.0101:1 against this white background. This clears the 3:1 minimum that applies to large-scale text under WCAG 1.4.3, so a correct gate must accept this page -- but only if it correctly detects the 26px/400-weight text as "large scale" and applies 3:1 rather than the normal-text 4.5:1 threshold, which this ratio would fail.
